# friction_long_term_alps
Code and processed data to reproduce the analyses and figures of the study "Constraining the glacier basal friction law from multidecadal to century scales observations of surface velocity and thickness changes on Alpine glaciers" (M. Zeller, A. Gilbert, F. Gimbert).

## structure

data/ processed datasets \n
src/ python scripts for preprocessing, analyses and figure generation

## data availability

Processed data used to generate the figures are provided in the folder 'data/processed'

The raw observational datasets are publicly accessible from their respective repositories:
The Elmer/Ice finite-element software used in this study is open source and available at https://github.com/ElmerCSC/elmerfem ([Gagliardini et al., 2013](https://doi.org/10.5194/gmd-6-1299-2013)), . 
Surface DEMs, velocity and elevation datasets for the Alpine glaciers analyzed in this study are available from the following sources: 
- GLAMOS database (https://www.glamos.ch); 
- Swiss glaciers monitoring programs ([Bauder, 2016](https://doi.glamos.ch/pubs/glrep/glrep_133-134.html); [Bauder et al., 2022](https://doi.glamos.ch/pubs/glrep/glrep_141-142.html)); 
- GLACIOCLIM database (https://glacioclim.osug.fr);
- French glaciers monitoring programs (Saint-Sorlin: [Vincent et al., 2000](https://doi.org/10.3189/172756500781833052); Argenti√®re: [Vincent et al., 2009](https://doi.org/10.3189/172756409787769500))
